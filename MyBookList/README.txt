1.) Install MongoDB (https://www.mongodb.com/download-center?jmp=nav#community)
2.) After installing MongoDB, go to Program Files/MongoDB/Server/<version>/bin
3.) Run mongod.exe first then run mongo.exe next. Both are located inside the bin folder.
4.) Open Windows Powershell and go to MyBookList's location
5.) Type "npm start" then ENTER. No need to install express since I already installed it before (see node_modules folder).

6.) Homepage is "#/home/<username>"
7.) Login page is EITHER "#/" OR "#/session/new"
